Instructions for Using the Adobe Online Marketing Suite Solution Files

This folder contains the solution files for the Data Warehousing Article.


Follow the steps below to set up the files for the article.

If you have any comments or problems with these files, please report them to the Adobe Developer forum:
https://developer.omniture.com/form > General Topic Forum

Supported users can contact their Adobe ClientCare rep with questions.

Setting up the article solution files:

1. In Eclipse, move these files to a new Java project. 
2. In the omtr_api_config.wsdd file, replace your Web Services User Name.
3. In the PWCallback.java file, replace your SiteCatalyst Shared Secret.
4. In the DataWarehouse.java file, replace your Report Suite Id, your contact name, your contact phone, your email address, your web services username, and your web services shared secret.
